Gifting Guide (BETA FEATURE)

Below you will find all the information you need to know about gifting with Fourthwall.

If you use JetBrains tools you can use sample http requests from here.
Webhooks can be found here.

1. How to enable Streamelements gifting​

1.1. Go to Twitch Gifting App in your Fourthwall admin dashboard.
1.2. Select Streamelements gifting radio button and ensure that your shop is connected with Streamelements.
1.3. Click on the Enable checkbox to enable the Twitch Gifting App.

2. Webhook configuration​

Go to Webhook Settings And create webhook for

  • GIFT_DRAW_STARTED
  • GIFT_DRAW_ENDED

3. How to start Gift Draw​

3.1 Ensure your stream is live.

PUT https://{{host}}/open-api/v1.0/streaming/start
Authorization: {{openApiUserAuth}}
Content-Type: application/json

{
  "services": [
    {
      "type": "TWITCH",
      "broadcasterId": "test-broadcaster-id",
      "broadcasterLogin": "raziosx",
      "thumbnailUrl": "test-thumbnail-url"
    }
  ]
}

3.2. Go to your shop main page and purchase as gift any product.

3.3. It should send webhook event on webhook Gift draw started configured in step 2.

4. How to finish Gift Draw​

4.1. Send a request to finish the draw.

PUT https://{{host}}/open-api/v1.0/gifting/draw/{{drawId}}/finish
Authorization: {{openApiUserAuth}}
Content-Type: application/json

{
  "participants": [
    {
      "service": "TWITCH",
      "userId": "{{twitchWinnerUserId}}",
      "userName": "{{twitchWinnerUserName}}"
    }
  ]
}

4.2. In response you will get the winner details along with redeem url.
4.3. We will also notify about the winner in the configured webhook Gift draw ended.

!!! Gift draws not finished in configured time (+ 1 minute) will be automatically finished without winners to allow redoing the draw.
