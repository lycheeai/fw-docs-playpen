Fields Description​

FieldDescription
testModeIf value is true, it indicates that the event contains only test data. See Testing paragraph for more info.
idIdentificator of the event, the value is always unique. If you receive multiple events with the same id, you should treat them as duplicates. See Limitations paragraph.
webhookIdIndicates which webhook configuration was used.
shopIdYour shopId, useful if you want to manage multiple shops under the same webhook endpoint.
typeIndicates which data type you should expect within the event.
apiVersionVersion of the data type that was chosen.
createdAtDate at which the event was created.
dataContains all the actual data specific to the type of the event. Refer to the Open API model to see available fields for types.
